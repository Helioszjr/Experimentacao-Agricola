Descrio

Este repositrio conta com diversas rotinas do software R voltadas para o auxilio de estudantes. Aqui aborda, experimentos e exercicios em Estatstica Experimental aplicada pesquisa agrcola. Delineamentos experimentais como DIC, DBC e parcelas subdivididas, alm de experincia com testes de significncia, anlise de regresso (incluindo polinmios ortogonais) e anlise de covarincia.
